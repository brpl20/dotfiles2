# Navegação Entre Desktops / WorksPaces 
1. Desktop Switchs
  [super] + [,<] / [.>] : Sempre vai ser uma especial e mais < > (,.) simulando setas de direita e esquerda. O que ocorre é que no Arch o uso das janelas já usa as setas do teclado: 

2. Desktop Move Windows { Mover a janela atual entre os desktops } 
[super] + [shift] + [alt (ao lado do espaço)] + [,<] / [.>] : Pegar a janela atual e mover para o desktop do lado

[super] + [,<] / [.>] : navegar entre abas 
[super] + [shift] + [,<] / [.>] : mover entre abas

[super] + w : sair do processo integralmente
[ctrl] + w : fechar aba

3. Move Tabs: 
A movimentacao UNIVERSAL entre abas deve ser:
- Ctrl + [pgUp] / [pgDown]  para todos os apps que possuem tabs
- Ctrl + Shift + [pgs] para mover entre os apps
- No terminal nao se seguro usar Ctrl + ] [ -> Scape
prev_tab = <Control><Shift>bracketleft                                                                                                                                                                                                   
next_tab = <Control><Shift>bracketright
